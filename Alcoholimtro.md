# Alcoholimetro

## Problematica
Detectar el color con base a la cantidad de alcohol que tiene el individuo
Exiten 4 colores
rojo 5.35 o mas
naranja 3.2 a 5.44
amarillo 2 a 3.2
verde de 0 a 1.9

## Pseudocodigo 

Inicio 
    leer nivel_alcohol 
    si ( nivel_alcohol >= 5.35 )
        Imprime Color rojo
    si no 
        si (nivel_alcohol >= 3.2 )
            Imprime Color Naranja
        si no 
            si ( nivel_alchol >= 2 )
                Imprime Color Amarillo
            si n0 
                si (nivel_alcohol >= 0)
                Imprime Color verde

Fin
